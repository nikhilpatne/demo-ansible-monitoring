These steps should be completed post deployment:

  1) Configure Zabbix
    a) Configuration -> Templates -> Template OS Linux
      i) Disable swap space check
    b) Configuration -> Actions
      i) change "Event Source" to "Auto Registration"
      ii) Create New Action:
          Under Action:
            Name: auto
          Under Operations:
            Operations: Add Host
                        Link to Template: Template OS Linux
                        Enable Host
          Add
    c) Hosts should now start appearing. Restart zabbix-agent on clients for fatser results
    d) Administration -> Users
      i) Change the Admin user password
    e) Administration -> User Groups
      ii) Disable guest user group
