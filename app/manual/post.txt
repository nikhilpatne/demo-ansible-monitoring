System Hostnames:
  Zabbix: zabbix.DEMO-PREFIX.R53-DOMAIN
  Ansible Tower: tower.DEMO-PREFIX.R53-DOMAIN
  App Node 0: app-0.DEMO-PREFIX.R53-DOMAIN
  App Node 1: app-1.DEMO-PREFIX.R53-DOMAIN
  App Node 2: app-2.DEMO-PREFIX.R53-DOMAIN

Default Credentials:
  Zabbix: Admin/zabbix
  Tower: admin/USER-DEFINED

These steps should be completed post deployment:

  1) Configure Zabbix
    a) Configuration -> Templates -> Template OS Linux
      i) Disable swap space check
    b) Configuration -> Actions
      i) change "Event Source" to "Auto Registration"
      ii) Create New Action:
          Under Action:
            Name: auto
          Under Operations:
            Operations: Add Host
                        Link to Template: Template OS Linux
                        Enable Host
          Add
    c) Hosts should now start appearing. Restart zabbix-agent on clients for fatser results
    d) Administration -> Users
      i) Change the Admin user password - must be the same as Tower admin password
    e) Administration -> User Groups
      ii) Disable guest user group
    f) Add App nodes to MySQL template
    g) fix MySQL is Down check
       {Template DB MySQL:mysql.ping.last(0)}=0 or {Template DB MySQL:mysql.ping.nodata(20)}=1
       Update interval to 14s
  2) Configure Ansible Tower
    a) ensure localhost is not in the inventory
    b) disable zabbix and tower in the inventory
